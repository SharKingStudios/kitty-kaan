## Some context

Its a simpler project for a simple idea.

The goal was to make the electronics stack as small as physically possible, which is why seperate breakouts were used. They also added to the simplicity of the design (which is nice).

Fitting everything together was the hardest part of this mini project, but it turned out looking pretty awesome!
